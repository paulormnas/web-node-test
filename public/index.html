LABORATÓRIO DE INFORMÁTICA

O Laboratório de Informática (Lainf) se encontra dentro da Divisão de Metrologia em Tecnologia da Informação e teleComunicações (Dmtic), do Inmetro. Dentre suas principais atividades destacam-se:

  • Contribuição no desenvolvimento da regulamentação brasileira para equipamentos metrológicos e ativos de Tecnoligia da Informação e teleComunicações (TIC)
  • Serviço de avaliação de produto de software para dispositivos metrológicos
  • Desenvolvimento de pesquisas relacionadas a segurança da informação, ensaio de proficiência para softwares, Internet das Coisas (IoT) e blockchain para metrologia

Monitoramento Ambiental

Este projeto tem por objetivo contribuir para o desenvolvimento de um sistema web integrado dentro do campus do Inmetro, de forma a conectar os dispositivos de monitoramento ambiental na área da Diretoria de Metrologia Científica (Dimci) e relacionar os dados de medições de modo a identificar eventos e contribuir para o controle do ambiente.

Este projeto utiliza o protocolo de RadNet, baseado em redes oportunísticas, na comunicação entre dispositivos, garantido a independência de uma infraestrutura de rede.
